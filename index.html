📁 BAGIAN 1: Membuat Website Pintar di Laragon

Langkah 1: Menyalakan Server

  • Buka aplikasi Laragon.
  • Klik tombol Start All sampai status Apache dan MySQL menyala.

Langkah 2: Membuat Rumah Proyek & Folder Ala GitHub

  • Klik kanan di area Laragon > pilih Quick App > pilih Blank.
  • Beri nama proyek: web-pertamaku lalu klik OK.
  • Klik tombol Root pada Laragon untuk membuka foldernya di komputer.
  • Masuk ke folder web-pertamaku, lalu buat dua folder baru di dalamnya agar rapi seperti repositori GitHub:
    • Folder css (untuk menghias tampilan).
    • Folder js (untuk menyimpan otak/logika).

Langkah 3: Membuat File Web

  • Di dalam folder utama web-pertamaku, klik kanan > New > Text Document.
  • Beri nama index.html (hapus akhiran .txt-nya).
  • Buka file tersebut menggunakan Notepad atau VS Code, lalu tempelkan kode web pintar ini dan simpan (Ctrl + S):
<!-- Kode HTML Web Menyapa Berdasarkan Waktu -->
(Gunakan kode index.html terakhir yang bisa memunculkan input nama dan tombol hijau)

Langkah 4: Uji Coba Web

  • Buka browser (Chrome/Edge), lalu ketik alamat: http://localhost/web-pertamaku/
  • Ketik nama Anda, klik tombol hijaunya, dan lihat keajaibannya!

🛢️ BAGIAN 2: Merancang Database di DBeaver

Langkah 1: Menyambungkan Jembatan Data

  1. Buka aplikasi DBeaver.
  2. Klik ikon Colokan Listrik Plus Hijau di pojok kiri atas.
  3. Pilih MySQL > klik Next.
  4. Pastikan Host: localhost, Username: root, dan kosongkan kolom Password.
  5. Jika diminta unduh driver, klik Download. Jika error, ganti versi driver di Driver Settings.
  6. Klik Test Connection. Jika muncul tulisan Connected, klik OK lalu klik Finish.

    7. Langkah 2: Merancang Tabel "pengunjung"

    1. Di sebelah kiri, klik panah pada localhost > klik panah database web-pertamaku (yang otomatis dibuatkan oleh Laragon).
    2. Klik kanan pada folder Tables > pilih Create New Table.
    3. Ubah nama tabel di bagian atas dari NewTable menjadi pengunjung.
    4. Buat Kolom 1 (id): Klik kanan di area tengah > Create New Column. Isi Name: id, Data Type: INT. Centang kotak Not Null dan Auto Increment.
    5. Beri Kunci Utama: Klik kanan pada baris id yang baru dibuat > pilih New Constraint from Selection > pastikan tipenya PRIMARY KEY > klik OK. (Akan muncul simbol kunci emas/tulisan PRI).
    6. Buat Kolom 2 (nama): Klik kanan lagi > Create New Column. Isi Name: nama, Data Type: VARCHAR(50). Klik OK.
    7. Klik tombol Save... di barisan paling bawah, lalu klik Persist.

    Langkah 3: Mengisi Data Sukses

    1. Klik tab Data di bagian atas layar tengah (di sebelah tab Properties).
    2. Klik ikon Garis Tabel dengan Tanda Plus Hijau (+) di barisan tombol paling bawah untuk menambah baris baru.
    3. Klik dua kali pada kolom nama, ketik Yudi, lalu tekan Enter.
    4. Klik tombol Save (ikon biru/disket di barisan bawah) untuk mengunci data. Kolom id akan otomatis berubah menjadi angka 1! Sukses!
